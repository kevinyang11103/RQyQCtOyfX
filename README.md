# Python+Django 图书管理系统 python671

## 项目概述

本系统基于Python语言和Django框架，主要用于管理图书信息及出版社相关数据。系统支持出版社管理、图书管理、作者管理和数据统计等功能，能够实现数据的增删改查操作，并具备自动更新数据功能。

## 技术栈

- **后端**: Python 3.x, Django
- **数据库**: SQLite（或其他支持的数据库系统）
- **开发工具**: PyCharm

## 功能模块

### 出版社管理

- 添加/修改/删除出版社信息

### 图书管理

- 图书信息录入/编辑/删除
- 图书检索

### 作者管理

- 作者信息录入/编辑/删除

### 系统管理

- 数据统计与展示
- 数据自动更新

## 系统角色

- 管理员：负责系统的日常管理和维护工作

## 运行环境

- 操作系统：兼容Python和Django的操作系统
- Python版本：3.x
- Django版本：本项目所使用版本

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 克隆或下载项目代码

具体步骤请参考项目提供的运行教程。
